### Задача 1
Создать класс Circle, конструктор которого принимает радиус. Класс должен иметь два метода, вычисляющие площадь и длину окружности.
Пример вызова класса:
```python
NewCircle = Circle(8)
print(NewCircle.area())
print(NewCircle.perimeter())
```
